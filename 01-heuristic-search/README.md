# Pràctica 1: Cerca Local


#### Compilar
Per compilar la pràctica simplement s'ha d'executar el 'make jar' a la terminar, dins de la carpeta practica1

#### Executar
Per executar el codi es recomana usar la comanda 'java -jar Practica1.jar num_experiment' a la terminal, una vegada compilat.

num_experiment és un número de l'experiment (existeixen 7 experiments, així que és un número del 1 al 7), per altra banda, es pot escriure hc o sa, per executar la verisó exemple de l'enunciat amb l'algorisme sel·leccionat. (hc -> hill climbing i sa -> Simulated anealing).



#### Exemple pas a pas d'execució (executar l'experiment 1):

1r Descomprimir

2n cd Practica1

3r make jar

4t java -jar Practica1.jar 1
